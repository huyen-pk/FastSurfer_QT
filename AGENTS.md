# Multi-Agent System Standards

This repository utilizes a multi-agent system to ensure high-quality software development and strict adherence to architectural standards.

## System Workflow

All feature development and bug fixes must follow the established multi-agent workflow:
- [Multi-Agent Workflow](.agents/workflows/multi-agent-workflow.md)

## Agent Roles

Each agent in the system has a specific role and set of responsibilities defined in `.agents/roles/`.

## Key Policies
- Before generating code, read the standards in `.agents/README.md`.
- When performing a task, identify your current role by checking the corresponding file in `.agents/roles/`.
- Strict adherence to the "Hard Blocks" in `.agents/workflows/multi-agent-workflow.md` is required.
